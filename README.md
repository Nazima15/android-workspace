# 📘 Kotlin for Java Developers

## 📌 프로젝트 소개

이 프로젝트는 **Java 개발자를 위한 Kotlin 기초 학습 및 실습**을 위한 저장소입니다.
Kotlin의 핵심 문법과 함수 개념을 간단한 코드 예제로 익히는 것을 목표로 합니다.

---

## 🎯 학습 목표

* Kotlin 기본 문법 이해
* Java와 Kotlin 차이 파악
* 함수(Function)와 람다(Lambda) 이해
* Scope Functions 활용

---

## 🧱 기술 스택

* Kotlin 2.x
* IntelliJ IDEA 또는 Android Studio

---


## 💡 Java vs Kotlin 차이

| Java    | Kotlin      |
| ------- | ----------- |
| 함수 제한적  | 함수 = 값처럼 사용 |
| 익명 클래스  | 람다          |
| verbose | 간결          |

---

## 📝 실습 예제

```kotlin
fun main() {
    val numbers = listOf(1, 2, 3)

    numbers.forEach {
        println(it * 2)
    }
}
```

---

## ✅ 핵심 정리

* `it` = 함수에 들어온 값
* `this` = 객체 자기 자신
* Kotlin = **함수를 값처럼 사용하는 언어**
